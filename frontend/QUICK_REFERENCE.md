# BlockNote Editor - Quick Reference

## 🚀 Quick Start

Type `/` anywhere in the editor to open the slash menu.

---

## 📋 Custom Blocks

| Command | Block | Use For |
|---------|-------|---------|
| `/metrics` | Financial Metrics | Display 8 key financial metrics in cards |
| `/table` | Data Table | Sortable tables with mini-charts |
| `/thesis` | Investment Thesis | Bull/Bear/Base case analysis |
| `/embed` | Embed | YouTube, Twitter, TradingView embeds |
| `/citation` | Citation | Research source citations |
| `/highlight` | Highlight | Important callouts with colors |
| `/company` | Company Mention | Tag companies with links |

## 💡 Pro Tips

### Financial Metrics Block
1. Click "Edit" to enter values
2. Values auto-format (currency, percentage)
3. Color-coded for positive/negative growth

### Data Table Block
1. Click column headers to sort
2. Choose column type for auto-formatting
3. Mini-charts show trends for numbers
4. Add/remove rows and columns in edit mode

### Investment Thesis Block
1. Use tabs to separate Bull/Bear/Base cases
2. Statistics show completion progress
3. Great for scenario planning

### Research Templates
1. Insert template first
2. Fill in your specific details
3. Delete sections you don't need
4. Customize structure as needed

## 📦 Block Features

### All Blocks Support
- **Auto-save**: Changes save automatically (2s delay)
- **JSON Storage**: Content stored as structured JSON
- **HTML Export**: Can export to HTML
- **Copy/Paste**: Works across documents

### Financial Metrics Block
- 8 metrics: Revenue, Growth, Net Income, Margin, P/E, P/S, D/E, ROE
- Edit mode for data entry
- Color-coded values

### Data Table Block
- Unlimited rows and columns
- 4 column types: text, number, currency, percentage
- Sortable by clicking headers
- Mini-charts for numeric columns

### Thesis Block
- 3 scenarios: Bull, Base, Bear
- Color-coded tabs
- Point counting per scenario

### Embed Block
- Auto-detects: YouTube, Twitter, TradingView
- Fallback to generic iframe
- Optional captions

## 🔧 Troubleshooting

### Slash Menu Not Showing
- Make sure you're typing `/` in the editor
- Try clicking in the editor first
- Refresh page if issues persist

### Blocks Not Saving
- Check network connection
- Look for save status indicator (top-right)
- Green ✓ means saved successfully

### Table Sorting Not Working
- Click directly on column header text
- Click again to reverse sort direction
- Only works in view mode (not edit mode)
